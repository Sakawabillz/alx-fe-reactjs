# GitHub User Search

A React application that allows users to search for GitHub profiles and view their information using the GitHub API.

## Features

- Search for GitHub users by username
- View user profile information including:
  - Avatar
  - Name and username
  - Bio
  - Join date
  - Number of repositories, followers, and following
  - Location, company, website, and Twitter handle (if available)
- Responsive design that works on mobile and desktop

## Prerequisites

- Node.js (v14 or later)
- npm (v7 or later) or yarn

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/github-user-search.git
   cd github-user-search
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- GitHub API
- Axios (for API requests)
